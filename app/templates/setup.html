Application Setup

Before you run gphotos_sync for the first time, you need to create your own OAuth client ID. Follow these instructions: Creating an OAuth Client ID.

Once the client ID is created, download it as client_secret.json and save it under the application configuration directory:

  • ~/Library/Application Support/gphotos-sync/ (Mac OS X)
  • ~/.config/gphotos-sync/ (Linux)
  • C:\Users\<username>\AppData\Local\gphotos-sync\gphotos-sync\ (Windows)

After setting up the client ID, you are ready to run gphotos-sync. The first run will require a user login. See User Login OAuth Flow.

Syncing Google Photos

To sync your Google Photos with the application, follow these steps:

  1. Open your command line interface (Terminal, Command Prompt, etc.).
  2. Navigate to the root directory of the application.
  3. Run the command gphotos-sync .\gallery\ to start the synchronization process.
  4. Follow any prompts in the command line to complete the sync.

This will sync your Google Photos to the 'gallery' folder in the application directory.
